ruta.pretrain {ruta}R Documentation

Pretrain autoencoders.

Description

Obtain initial weights for an autoencoder via stack of RBMs.

Usage

ruta.pretrain(x, task, epochs = 10, ...)

Arguments

x

A "rutaAutoencoder" object.

task

A "rutaTask" object.

epochs

The number of epochs for the training process of each RBM.

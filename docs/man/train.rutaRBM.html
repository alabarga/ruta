train.rutaRBM {ruta}R Documentation

Train a Restricted Boltzmann Machine.

Description

Train a Restricted Boltzmann Machine.

Usage

## S3 method for class 'rutaRBM'
train(x, task, ...)

Arguments

x

A "rutaRBM" object. This is the learner that will be trained.

task

A "rutaUnsupervisedTask" object. It contains the data that will be used to train the RBM. Class information, if present, will not be used.

...

Additional parameters for the training function

Value

A "rutaModel" object containing the trained model.

train.rutaAutoencoder {ruta}R Documentation

Train an autoencoder.

Description

Train an autoencoder.

Usage

## S3 method for class 'rutaAutoencoder'
train(x, task, ...)

Arguments

x

A "rutaAutoencoder" object. This is the learner that will be trained.

task

A "rutaUnsupervisedTask" object. It contains the data that will be used to train the autoencoder. Class information, if present, will not be used.

...

Additional parameters for the MXNet optimizer

Value

A "rutaModel" object containing the trained model.

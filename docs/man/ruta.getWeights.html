ruta.getWeights {ruta}R Documentation

Get weights from any layer from a trained model.

Description

Get weights from any layer from a trained model.

Usage

ruta.getWeights(model, layer)

Arguments

model

A "rutaModel" object.

layer

An integer indicating the index of the layer for weights to be extracted from.

Value

A matrix containing layer weights.

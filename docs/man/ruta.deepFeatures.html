ruta.deepFeatures {ruta}R Documentation

Get the deep features from a trained autoencoder model.

Description

Get the deep features from a trained autoencoder model.

Usage

ruta.deepFeatures(model, task, ...)

Arguments

model

A "rutaModel" object from an autoencoder learner.

task

A "rutaTask" object.

...

Custom parameters for internal prediction function.

Value

A matrix containing the encoding output for each instance in the given task.

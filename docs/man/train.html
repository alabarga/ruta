train {ruta}R Documentation

Train a learner object.

Description

train creates a trained model for a given learner.

Usage

train(x, ...)

Arguments

x

Learner (possibly a "rutaLearner" object).

...

Parameters passed to the specific method.

Details

This is a generic function, will find a specific method for each learner class.

Value

A trained model.

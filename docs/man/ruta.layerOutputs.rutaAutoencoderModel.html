ruta.layerOutputs.rutaAutoencoderModel {ruta}R Documentation

Get outputs from any layer from a trained model and new data.

Description

Get outputs from any layer from a trained model and new data.

Usage

## S3 method for class 'rutaAutoencoderModel'
ruta.layerOutputs(model, task, layerInput = 1,
  layerOutput, ...)

Arguments

model

A "rutaModel" object from an autoencoder learner.

task

A "rutaTask" object.

layerInput

Currently input is always injected into the first layer.

layerOutput

An integer indicating the index of the layer to be obtained.

...

Custom parameters for internal prediction function.

Value

A matrix containing layer outputs for each instance in the given task.

ruta.layerOutputs.rutaRBMModel {ruta}R Documentation

Get outputs from any layer from a trained model and new data.

Description

Get outputs from any layer from a trained model and new data.

Usage

## S3 method for class 'rutaRBMModel'
ruta.layerOutputs(model, task, layerInput = 1,
  layerOutput = 2 - layerInput, ...)

Arguments

model

A "rutaRBMModel" object trained with an RBM learner.

task

A "rutaTask" object.

layerInput

Just 1 or 2 (default = 1).

layerOutput

The remaining layer (optional).

...

Custom parameters for internal prediction function.

Value

A matrix containing layer outputs for each instance in the given task.

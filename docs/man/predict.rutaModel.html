predict.rutaModel {ruta}R Documentation

Predict outputs for trained models and new data.

Description

You should pass a model argument and a task argument at least. The task can be the same input data the model was trained with, or new data with the same shape. Other arguments will be passed to the internal prediction function.

Usage

## S3 method for class 'rutaModel'
predict(object, ...)

Arguments

object

A "rutaModel" object.

...

Custom parameters for MXNet prediction function.

Value

A matrix containing predictions for each instance in the given task.

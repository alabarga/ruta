ruta.makeTask {ruta}R Documentation

Create a representation for a learning task.

Description

Create a representation for a learning task.

Usage

ruta.makeTask(id, data, cl = NULL)

Arguments

id

A string. A name for the task.

data

A dataset, will be casted to data.frame with as.data.frame.

cl

The index of the column corresponding to the class, or NULL if the dataset has no class.

Value

A generic task object with the provided data and parameters.

ruta.makeLearner {ruta}R Documentation

Create a representation for a learning algorithm.

Description

Create a representation for a learning algorithm.

Usage

ruta.makeLearner(cl, id = cl, ...)

Arguments

cl

A string. Type of learner.

id

A string. A name for the learner.

...

Additional parameters for the learner.

Value

An object for the learner containing the provided parameters.

Examples

ruta.makeLearner("rbm", hidden = 4, activation = "sigm")
## # ruta Learner
## # Type: RBM
## # Backend: deepnet
## # Activation: sigm
